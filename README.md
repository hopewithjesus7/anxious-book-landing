# 크리스천도 불안할 수 있습니다 — 소개 랜딩페이지

정적 단일 페이지 사이트 (HTML/CSS/JS, 백엔드 없음). 표지 파스텔 톤(하늘→라벤더→피치) 기반의 모던·고급 디자인.

## 보기 / 실행
- 그냥 `index.html`을 브라우저로 열면 됩니다.
- 로컬 서버로 보려면(권장): `python -m http.server 8000` 실행 후 http://localhost:8000
- 배포: Netlify / Vercel / GitHub Pages 에 `website/` 폴더 그대로 올리면 됩니다 (전부 무료).

## 폴더 구조
```
website/
  index.html
  css/style.css
  js/main.js          # 네비·스크롤 애니메이션·자가진단 로직
  assets/img/         # 웹 최적화된 이미지 (build_assets.py로 생성)
  build_assets.py     # 원본(../assets) → 웹용 이미지 재생성 스크립트
```
이미지 재생성: `python build_assets.py`

## 구성 섹션
히어로 → 이런 마음(공감 + 통계 시각화) → 선언 배너 → 세 가지 돌봄(컬러 그레이딩) → 자가진단(빈도 척도) → 저자 3인 → 추천사(원형 프로필) → 관계기술훈련 → 구매 → 푸터(연락처) + 문의 모달

## 인터랙션
- **자가진단**: 책 1장 6문항을 빈도 척도(0~3점)로 재구성. 합산 0~18점 → 안정/경도/중등도/높음 4단계 결과 + 점수 미터 + 면책 문구.
- **문의 모달**: "문의·신청하기"(관계기술훈련 섹션·푸터) 클릭 → 이름/연락처/이메일/관심항목/문의내용 입력 폼 → 제출 시 완료 화면.

## ⚠️ 시연용 → 실제 운영 시 교체할 항목
1. **연락처(이메일·전화)** — 푸터에 `XXXX@XXXX.com` / `02-XXXX-XXXX` 형태의 **마스킹 placeholder**로 들어가 있음. 실제 값으로 교체.
2. **문의 폼 전송** — 현재는 시연용으로 **실제 전송되지 않고** 완료 화면만 표시. 실제 접수하려면 `js/main.js`의 폼 submit 부분을 Formspree / Google Forms / 자체 백엔드로 연결.
3. (선택) 관계기술훈련 일정·대상 등 상세 텍스트 보강.

## 권리 관련 메모
- 책 표지·저자 사진·추천사는 저작권/초상권 대상입니다. 배포 전 **이기원 목사님(및 두란노) 동의** 확인 권장.
- 구매는 직접 결제 없이 공식 서점(두란노몰·교보·예스24·갓피플몰) 링크로만 연결됩니다.
- 자가진단에는 "의학적 진단이 아님" 면책 문구가 포함되어 있습니다.
